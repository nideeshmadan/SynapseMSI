# Synapse Research Layer

This directory contains institutional-grade research outputs from Synapse's market-state integrity analysis framework.

## Quickstart: Run the Integrity Validator

```bash
pip3 install -r requirements.txt
python3 synapse_validate.py client_exports/dataset_preview_v1/sample_data/btc_apr09_sample.parquet
```

The validator analyzes already-reconstructed market states and validates temporal coherence using strict policy constraints. This tool does not fetch exchange data or generate trading signals—it validates existing data quality.

**Synapse Temporal Coherence Policy:**
- `sync_gap_ms <= 500` (cross-venue synchronization)  
- `best_pair_age_ms <= 1000` (quote freshness)

States failing either constraint are marked as temporally invalidated and unsuitable for replay applications or execution simulation.

Collect live public BTCUSDT perpetual data and validate temporal coherence:

```bash
# Step 1: Collect live snapshots (5 minutes)
python3 collect_public_binance_bybit.py --seconds 300

# Step 2: Build cross-venue states
python3 build_public_crossvenue_states.py

# Step 3: Validate temporal coherence  
python3 synapse_validate.py public_data/btc_binance_bybit_states.csv
```

**Important Note:** This public demo uses live REST polling and collection-time timestamps where venue event timestamps are unavailable. The institutional research package uses preserved venue-side timestamps from Synapse canonical reconstruction, so this public demo is for workflow demonstration, not a replacement for full canonical replay.

## Purpose

Synapse Research provides quantitative analysis of temporal coherence in cross-venue perpetual futures markets, focusing on:

- **Market Data Quality Assessment**: Systematic evaluation of cross-venue synchronization gaps and quote freshness
- **Execution Simulation Validation**: Temporal validity frameworks for backtesting and market-making simulation
- **Signal Integrity Analysis**: Detection and classification of phantom arbitrage opportunities arising from temporal misalignment
- **Regulatory Documentation**: Comprehensive market data quality reporting for institutional compliance
